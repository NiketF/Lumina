# Lumina — AI-Generated Landing Page

## Overview
Lumina is an AI-generated landing page built using prompt engineering as part of a competition hosted by Outlier.ai.

The project focuses on leveraging Large Language Models (LLMs) to design and generate a complete frontend interface by structuring and refining prompts, rather than manually building the UI from scratch.

## Problem Statement
In fast-paced environments like hackathons or product ideation, building visually structured landing pages from scratch is time-consuming.

This project explores:
- How effectively AI can generate usable UI
- How prompt engineering can control layout, structure, and content
- How quickly a functional prototype can be built using LLMs

---

## Solution Approach
- Designed structured prompts to guide the AI in generating:
  - Page layout
  - Section hierarchy
  - Content flow
  - Basic UI/UX elements
- Iteratively refined prompts based on output quality
- Validated usability and visual consistency of the generated page

---

## Key Features
- AI-generated landing page with structured sections
- Prompt-driven UI generation workflow
- Rapid prototyping without manual frontend development initially
- Clean and readable layout with basic UI components

## Tech Stack
- HTML, CSS, JavaScript (AI-generated output)
- Prompt Engineering (LLM-based generation using Claude)
